How It Works

MiniRAG uses a lightweight pipeline to process questions and provide accurate answers.

1

Data Processing

Game transcripts are chunked.

2

Embedding Generation

Converted using Sentence Transformers.

3

Vector Storage

Stored in FAISS or Chroma.

4

Query Processing

Semantic search runs over vectors.

5

Answer Generation

Local LLM generates the reply.

Installation & Setup

Requires Python 3.10+ and at least 4GB RAM.

# Clone the repository git clone https://github.com/Akki-Maharaj/Genshin-MiniRAG-Offline-Assistant-with-Local-LLM.git cd Genshin-MiniRAG-Offline-Assistant-with-Local-LLM # Install dependencies pip install -r requirements.txt # Run the application python app.py

System Requirements

  • Python 3.10 or higher
  • 4GB RAM minimum (8GB recommended)
  • 2GB free disk space
  • GPU optional but recommended
